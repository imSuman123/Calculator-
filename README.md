# iOS Calculator (SwiftUI)

A simple, modern iOS calculator built with SwiftUI.

## Features
- Basic arithmetic: add, subtract, multiply, divide
- Clear, percentage, and sign toggle
- Responsive layout using SwiftUI grids

## Requirements
- Xcode 15+
- iOS 17+ (can be lowered in the project settings if needed)

## Getting Started
1. Open `CalculatorApp.xcodeproj` in Xcode.
2. Select a simulator or device.
3. Build and run.

## Project Structure
- `CalculatorApp/CalculatorAppApp.swift`: App entry point
- `CalculatorApp/ContentView.swift`: UI layout
- `CalculatorApp/CalculatorLogic.swift`: Calculator state & operations
- `CalculatorApp/CalculatorButton.swift`: Button styling
